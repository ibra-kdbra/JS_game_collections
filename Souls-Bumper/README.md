# Souls Bumper

A vanilla JavaScript platformer game created for the JS13k Game Jam #11.

## Theme

**Death**

## Description

Souls Bumper is a platformer where you navigate through challenging levels. Use your unique abilities to overcome obstacles and reach the goal.

## Controls

- **[A][D]** or **[←][→]**: Move Left / Right
- **[W]**, **[Space]**, or **[↑]**: Jump
- **[E]**: Rewind Ghost
